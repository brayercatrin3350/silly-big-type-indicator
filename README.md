1. Right-click tray icon → **"Settings"**
2. Click **"Appearance"** tab
3. Adjust:
   - **Font** — Select font and size
   - **Text Color** — Pick color or gradient
   - **Background** — Color or transparent
   - **Border** — Width, color, radius
   - **Shadow** — Enable/disable glow
4. Click **"Preview"** to test
5. Click **"Save"** when happy

### How to Change Animation Effect

1. Right-click tray icon → **"Settings"**
2. Click **"Animation"** tab
3. Choose effect:
   - **Fade** — Smooth fade in/out
   - **Slide** — Slides from edge
   - **Bounce** — Bouncy pop effect
   - **Typewriter** — Letters appear one by one
   - **Explosion** — Burst from center
4. Adjust duration (0.5-5 seconds)
5. Preview effect
6. Click **"Save"**

### How to Set Up for Streaming (OBS)

1. In tool settings, click **"Stream"** tab
2. Enable **"Browser Source Mode"**
3. Note the local URL (e.g., `http://localhost:12345`)
4. In OBS:
   - Add new **"Browser Source"**
   - Set URL to the local address
   - Set Width to 1920, Height to 1080
   - Check **"Refresh browser when scene becomes active"**
5. Click **"OK"**
6. Keystrokes now appear on stream!

### How to Use Key History Mode

1. Right-click tray icon → **"Settings"**
2. Click **"History"** tab
3. Enable **"Show Key History"**
4. Choose position (Top-Right, Bottom-Left, etc.)
5. Set max items (5, 10, 20)
6. Choose direction: Horizontal or Vertical
7. History panel appears showing recent keys
8. Press keys to see them added to history

### How to Enable Sound Effects

1. Right-click tray icon → **"Settings"**
2. Click **"Sounds"** tab
3. Enable **"Key Press Sounds"**
4. Choose sound pack:
   - **Mechanical Blue** (clicky)
   - **Mechanical Red** (smooth)
   - **Typewriter** (classic)
   - **Custom** (use your own WAV files)

6. Test sound with **"Play Test"** button
